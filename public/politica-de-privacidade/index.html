POLÍTICA DE PRIVACIDADE Oct 23 2025

by cmi

O endereço do nosso site é: http://localhost:8000/.

Pode pedir para receber um ficheiro de exportação com dados pessoais guardados ou publicados sobre si, incluindo qualquer dado pessoal que indicou. Também pode solicitar que os dados guardados sejam eliminados.

Ao carregar imagens para o site, deve evitar carregar imagens com dados incorporados de geolocalização (EXIF GPS). Os visitantes poderiam descarregar e extrair os dados de geolocalização das imagens do site.

Este site não guarda IP’s dos utilizadores. Não utiliza tracking para quaisquer fins. Não guarda dados durante mais do que o tempo necessário para a sua funcionalidade.

O site tem funcionalidades anti-spam que podem interferir com a utilização do mesmo, que está especificamente apontada a comentários, pelo que comentários com hiperligações poderão acabar na lista de spam.

Apesar de o software wordpress guardar os IP’s dos comentadores, utilizamos um plugin que permite que o website não guarde esse registo

Os artigos neste site podem incluir conteúdo incorporado (por exemplo: vídeos do youtube, imagens, artigos, etc.). É o caso de vídeos do YouTube. O conteúdo incorporado de outros sites comporta-se tal como se o utilizador visitasse esses sites.

Há um banner a proteger os visitantes de abrir carregar esse conteúdo automáticamente, sendo que a consulta de informação incorporada é opt-in.

Ao interagir com este website no fediverso via @imc@indymedia.pt, no Mastodon ou noutras redes ligadas por ActivityPub, as interações com o website são públicas, como seguir, “gostos” e comentários. Esta funcionalidade é apenas ativada caso o utilizador interaja com o website no fediverso, e é uma funcionalidade expectável das redes federadas. Estamos a aguardar um patch que permitirá esconder a lista de seguidores do website do público no futuro.
